# Générateur de citations (Anglais)

## Description
Ce projet est un **Générateur de citations développé avec React.js**.  
Il permet d’afficher des citations aléatoires et de générer une nouvelle citation au clic.  
Le projet est conçu avec un **design moderne** et responsive pour un rendu agréable sur mobile et desktop.

---

## Fonctionnalités
- Affichage d’une citation aléatoire
- Bouton pour générer une nouvelle citation
- Citation stylée avec auteur
- Design moderne et responsive
- Utilisation d'une **API publique** comme [Quotable](https://api.quotable.io/random)
